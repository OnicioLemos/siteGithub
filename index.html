Github

O GitHub é uma plataforma de hospedagem de código-fonte e controle de versão usando o sistema Git. Ele permite que os desenvolvedores colaborem em projetos, compartilhem código e gerenciem versões de software. O GitHub oferece recursos como repositórios, pull requests, issues e integração contínua, facilitando a colaboração e o desenvolvimento de software em equipe.
